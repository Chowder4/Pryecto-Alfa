# SOCIALGAMES

SOCIALGAMES es una plataforma donde todos los videojugadores pueden conectarse para conocer nuevas personas
con sus mismos gustos, jugar torneos y ganar variadas recompensas.

Este proyecto está realizado sobre php y html. Para los html se ha utilizado la tecnología de Tailwind,
un editor de estilos que cambia la forma tradicional de CSS por un sistema de clases que te permite añadir
estilos a la página. Para php se ha utilizado Php my admin para editar las bases de datos y XAMPP para 
abrir los servidores en localhost.

Esta idea nació con el objetivo de unir a la comunidad de jugadores. La plataforma tiene una página principal
en la que se muestran diferentes juegos, streams más vistos de la comunidad y los torneos que están próximos a 
comenzar. 
Hay dos tipos de torneos: los creados por el usuario para realizar competiciones entre amigos y/o desconocidos y 
los creados por los desarrolladores, que otorgaran diferentes premios a los ganadores de los mismos.
El usuario tambien es capaz de añadir videojuegos a la base de datos tanto videojuegos ya existentes como
juegos que haya creado el propio usuario para darle promoción y que otros jugadores lo prueben y 
suban en el ranking de dicho videojuego o ganen un torneo que se cree sobre él.

Este proyecto también deja a los usuarios crear su propio equipo, una subcomunidad que permite acceder 
al grupo para hablar con los integrantes de forma más rápida.

En el perfil, el jugador puede ver los equipos, torneos y juegos en los que está participando.
